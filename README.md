# Kompak Cipta

Proyek ini adalah sebuah aplikasi sederhana yang mencetak kata "Kompak", "Cipta", atau "Kompak Cipta" berdasarkan aturan tertentu. Aplikasi ini ditulis dalam bahasa PHP dan dapat dijalankan di server Apache menggunakan XAMPP.

## Persyaratan

- [XAMPP](https://www.apachefriends.org/index.html) harus terinstall pada komputer Anda.
- Git (opsional, jika ingin mengkloning repository dari GitHub)

## Instalasi

1. **Clone Repository dari GitHub (opsional)**

   Jika Anda belum mengkloning repository, buka terminal atau command prompt dan jalankan perintah berikut:
   ```bash
   git clone https://github.com/username/kompak-cipta.git
2. Pindahkan File ke Direktori htdocs XAMPP
   Jika Anda mengkloning repository atau mengekstrak file proyek, pindahkan semua file ke direktori htdocs di instalasi XAMPP Anda. Misalnya:
   ```bash
   C:\xampp\htdocs\kompak-cipta
## Menjalankan Proyek
1. Jalankan XAMPP
   Buka XAMPP Control Panel dan pastikan Apache server berjalan. Klik tombol "Start" di sebelah Apache untuk menjalankan server.

2. Akses Aplikasi di Browser
   Buka browser dan akses aplikasi dengan URL berikut:
   ```bash
   http://localhost/kompak-cipta/kompakcipta.php
Anda akan melihat form input untuk memasukkan jumlah perulangan. Masukkan angka dan klik "Submit" untuk melihat hasilnya.
